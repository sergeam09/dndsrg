📋 CÓMO OBTENER EL LINK
🎮 Discord (recomendado)
1. Sube la imagen a cualquier canal
2. Haz click en la imagen para abrirla
3. Click en "Abrir en el navegador" o en el ícono de expandir
4. Copia la URL — debe contener cdn.discordapp.com
✗ NO uses el link del canal (discord.com/channels/...)
✓ USA el link CDN (cdn.discordapp.com/attachments/...)
📁 Google Drive
1. Sube la imagen → click derecho → Compartir
2. Cambia a "Cualquier persona con el enlace"
3. Copia el link y pégalo aquí
✓ Se convierte automáticamente al formato correcto
🐙 GitHub
Sube la imagen a tu repo → click en ella →
botón Raw → copia esa URL
